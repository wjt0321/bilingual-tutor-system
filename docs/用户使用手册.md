## 欢迎使用双语导师系统

双语导师系统是一个专为中文用户设计的个性化语言学习平台，帮助您在2年内从CET-4英语和N5日语水平提升至CET-6+和N1+水平，每日学习时间仅需1小时。

## 快速开始

### 1. 系统访问

- **网页访问**: 在浏览器中打开 `http://localhost:5000`
- **健康检查**: 访问 `http://localhost:5000/health` 确认系统正常运行

### 2. 首次使用

1. **注册账户**: 点击"注册"按钮创建新账户
2. **设置个人信息**: 填写学习目标和当前语言水平
3. **完成初始评估**: 系统将评估您的英语和日语水平
4. **制定学习计划**: 系统自动生成个性化学习计划

## 主要功能介绍

### 🎯 个性化学习计划

系统根据您的语言水平、学习目标和可用时间，自动生成个性化的学习计划：

- **每日1小时学习时间分配**
- **20%时间用于复习，80%时间学习新内容**
- **英语和日语学习时间智能分配**
- **根据学习进度动态调整计划**

### 📚 智能内容推荐

系统会自动为您推荐适合的学习内容：

- **英语内容**: CET-4到CET-6级别的文章、视频、音频
- **日语内容**: N5到N1级别的学习材料
- **内容来源**: BBC Learning English、VOA、NHK等权威来源
- **质量保证**: 所有内容经过质量评估和难度分级

### 📊 学习进度跟踪

实时监控您的学习进展：

- **词汇掌握情况**: 跟踪已学词汇和掌握程度
- **技能提升曲线**: 听说读写各项技能的进步轨迹
- **目标达成进度**: 距离CET-6和N1目标的完成度
- **学习统计**: 学习时间、完成练习数量等统计信息

### 🔍 弱点智能分析

系统会分析您的学习弱点并提供针对性建议：

- **错误模式识别**: 分析常见错误类型
- **薄弱环节定位**: 识别需要加强的语法点和词汇
- **改进建议**: 提供具体的学习建议和练习推荐
- **优先级排序**: 按重要性排列需要改进的方面

### 🧠 科学复习机制

基于艾宾浩斯遗忘曲线的智能复习系统：

- **间隔重复算法**: 在最佳时机安排复习
- **复习内容优化**: 重点复习容易遗忘的内容
- **复习效果跟踪**: 监控复习效果并调整策略
- **长期记忆巩固**: 确保知识点转入长期记忆

## 详细使用指南

### 学习界面操作

#### 开始学习会话

1. 登录系统后，点击"开始学习"按钮
2. 系统会显示今日的学习计划
3. 选择要学习的语言（英语或日语）
4. 按照系统推荐的顺序完成学习活动

#### 学习活动类型

**阅读理解**
- 阅读推荐文章
- 回答理解问题
- 学习新词汇和短语

**听力练习**
- 收听音频材料
- 完成听力理解题
- 跟读练习提高发音

**词汇学习**
- 学习新单词和短语
- 记忆词汇含义和用法
- 完成词汇练习题

**语法练习**
- 学习语法规则
- 完成语法填空题
- 纠正常见语法错误

#### 学习进度保存

- 系统会自动保存您的学习进度
- 可以随时暂停和继续学习
- 学习记录会同步到云端（如果启用）

### 进度查看

#### 学习仪表板

访问主页面查看学习概况：

- **今日学习进度**: 已完成的学习时间和活动
- **本周学习统计**: 一周内的学习情况汇总
- **目标达成情况**: 距离CET-6和N1目标的进度
- **近期成就**: 最新获得的学习成就和里程碑

#### 详细进度报告

点击"进度报告"查看详细分析：

- **技能发展曲线**: 各项语言技能的提升轨迹
- **词汇掌握统计**: 已学词汇数量和掌握程度分布
- **学习效率分析**: 学习时间利用效率和改进建议
- **弱点分析报告**: 当前需要重点改进的方面

### 设置和个性化

#### 学习偏好设置

在"设置"页面可以调整：

- **学习时间**: 每日学习时间长度（建议1小时）
- **语言比重**: 英语和日语的学习时间分配
- **难度偏好**: 内容难度的偏好设置
- **学习提醒**: 设置学习提醒时间

#### 目标管理

- **短期目标**: 设置周目标和月目标
- **长期目标**: 调整2年学习目标
- **里程碑设置**: 自定义学习里程碑
- **奖励机制**: 设置达成目标后的奖励

## 学习建议和技巧

### 最佳学习实践

1. **保持规律**: 每天固定时间学习，养成习惯
2. **专注学习**: 学习时关闭干扰，专心致志
3. **及时复习**: 按照系统安排及时复习已学内容
4. **主动练习**: 积极完成系统推荐的练习活动
5. **记录笔记**: 记录重要知识点和个人心得

### 学习效率提升

- **预习准备**: 学习前简单预习即将学习的内容
- **总结回顾**: 学习后总结当天的收获和疑问
- **实际应用**: 尝试在日常生活中使用所学知识
- **交流练习**: 寻找机会与他人练习口语对话
- **持续反思**: 定期反思学习方法和效果

### 常见问题解决

**学习进度缓慢**
- 检查学习时间是否充足
- 确认学习内容难度是否合适
- 考虑调整学习策略和方法

**内容理解困难**
- 降低内容难度设置
- 增加基础知识学习时间
- 寻求额外的学习资源

**记忆效果不佳**
- 增加复习频率
- 使用多种记忆方法
- 确保充足的睡眠和休息

## 学习成功案例

### 典型学习路径

**初级用户（CET-4/N5水平）**
- 第1-3个月：巩固基础，建立学习习惯
- 第4-8个月：稳步提升，扩大词汇量
- 第9-15个月：强化练习，提高应用能力
- 第16-24个月：冲刺目标，达到CET-6/N1水平

**学习时间分配建议**
- 英语：35分钟（词汇15分钟，阅读10分钟，听力10分钟）
- 日语：25分钟（词汇10分钟，语法8分钟，听力7分钟）
- 复习：12分钟（英语8分钟，日语4分钟）

### 成功要素

1. **坚持不懈**: 每天坚持学习，不间断
2. **科学方法**: 遵循系统推荐的学习方法
3. **及时调整**: 根据学习效果调整策略
4. **积极心态**: 保持积极的学习态度
5. **实际应用**: 将所学知识应用到实际生活中
